# Stockfish Engine Binary

The Stockfish engine binary (`stockfish.exe`, ~110MB) is **not tracked in this GitHub repository** due to its size. It has been uploaded separately to Google Drive.

**Download link: [drive.google.com/file/d/19MXyRPApNBgSk2MraFqzUPXWP_0q1YJ3/view?usp=sharing](https://drive.google.com/file/d/19MXyRPApNBgSk2MraFqzUPXWP_0q1YJ3/view?usp=sharing)**

## Setup

After downloading, place `stockfish.exe` directly inside this folder (`backend/models/`) so it matches the path expected by the code.
